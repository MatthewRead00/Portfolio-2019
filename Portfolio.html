Bootstrap Online Store

Bootstrap Online Store

The purpose of this project was to create an online store using the Bootstrap framework. I decided to create a online electronics store for the sake of this project in which would sell such items as a Playstation 4 Pro console and an Apple Watch. On the homepage, you would be able to see a list of the six feature products in which would be sold through the store, each list item would show a picture of the product, as well as the product title, product price, and a "View" button, in which would take the user to a product modal in which would provide additional information regarding such product.

Project Date:

April 2019

Language(s)/Software:

HTML, CSS, JavaScript, Bootstrap
